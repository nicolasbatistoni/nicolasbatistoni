# Soy Nicolás Batistoni 👋

Software engineer con **15+ años** construyendo productos de punta a punta, desde Balcarce,
Argentina. Trabajo **full-stack**, **DevOps / cloud** y **seguridad ofensiva** por igual, y
diseño y entrego software como **[Otara Labs](https://otaralabs.com)**, mi estudio.

## 🧐 Acerca de mí

Me muevo en todo el stack: del modelo de dominio a producción, de la web y el mobile a los
motores de juego, la IA/ML y la robótica, con la CI/CD y la infraestructura que lo sostiene.
Certificado **eJPT** (INE Security), llevo una mirada de constructor a la seguridad ofensiva:
sé dónde suelen esconderse las fallas porque me tocó escribir y operar los sistemas.
